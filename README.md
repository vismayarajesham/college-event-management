# College Event Management System

A Flask-based web application developed to simplify the management of college events. The system allows students to register for events and enables administrators to manage events and registrations through a dedicated dashboard.

## Features
- Student Signup & Login
- Admin Login
- Event Creation & Management
- Individual Event Registration
- Group Event Registration
- Certificate Download
- Forgot Password (OTP via Email)
- Responsive User Interface

## Technologies Used
- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript

## Project Structure
```
college-event-management/
├── app.py
├── templates/
├── static/
├── requirements.txt
└── database.py
```

## Installation
1. Clone the repository
2. Install dependencies:
```
pip install -r requirements.txt
```
3. Run the application:
```
python app.py
```
